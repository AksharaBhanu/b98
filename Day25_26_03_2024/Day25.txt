Steps to convert MTC & Automation Script

1. execute the test case manually--System Study
2. while executing test case note down -
	page 		element 	action
	LoginPage	Username	enter
	LoginPage	password	enter
	LoginPage	go 		click
	HomePage	logout		verify present
3. for each page create respective POM Class  (ends with Page)

	a. declaration-->@FindBy
	b. intialization---> PageFactory
	c. utilization--> getters/setters


method name should be readable	
by default method should be public & static 
if it access atleast 1 non-static member of the class then it should be non-static
do not throw exception- handle it
in catch block print statcktrace
do not use any hardcoded value-use arg
Do not write Assert in POM class, it should be in TestNG class
verification method should return boolean value


4. create Test Class
  1. create a class inside script package, Name should be same as respective TestCase ID /Name
  2. every Test Class shoule extend BaseTest class
  3. we should write Test Method inside Test class (name should start with test) with priority
  4. specify manual test case steps as in-line comment
  5. below each comment , call respective POM class method
