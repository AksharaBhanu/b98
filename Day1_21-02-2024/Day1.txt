Selenium installtion
1. JDK 11 & above
2. IDE- Eclipse
3. selenium software
         
1. open Eclipse
2. select new workspace
3. create new Java Project
4. create a package (day1) under src
5. create a Demo1.java with main + sop
6. execute it and ensure that it is working fine
7. create a new folder 'lib' under the project
8. copy selenium jar file and paste it into lib folder
9. right click on selenium jar file-> build path->add to build path

 
How to see the source code of selenium
1. note down the FQCN -->org.openqa.selenium.chrome.ChromeDriver
2. goto https://github.com/SeleniumHQ/selenium/tree/trunk/java
3. click src->org/openqa/selenium
4. click chrome folder
5. click on ChromeDriver.java
