before performing any action( click type select double click dragdrop) we should find the element

How do find the element?
using findElement method with locator

What is locator?
selenium mechanism to identify the element

How many locators?
8

what are those?
1tagName
2id
3name
4className

5linkText
6partialLinkText

7cssSelector
8xpath

where are these locators present?
in By class

Note: 
By is an abstract class
all the locators are static method, so we use class name

(locator is duplicate)
if the locator value is matching with multiple elements then findElement method 
will return address of the 1st matching element

if the locator is not matching with any of the element then?
findElement method will throws NoSuchElementException

when do we use partialLinkText?
if the text of the link is
1. very lengthy or
2. dynamic 

linkText and partialLinkText- can be used to locate only hyperlink , if we try to locate any other type we get NSEE
