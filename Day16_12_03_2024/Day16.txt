Encapsulation
binding data member with member function so that it act as a single unit

3 steps
declaration
initialization
utilization

Encapsulation in selenium
class B
{
	private WebElement unTB;
	
	B(WebDriver driver)
	{
		unTB = driver.findElement(By.name("username"));
	}
	
	public void setUserName()
	{
		unTB.sendKeys("admin");
	}
}
